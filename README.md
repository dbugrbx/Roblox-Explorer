# Roblox-Explorer
An open source clone of the **Explorer Panel** from Roblox Studio.

## About
Roblox-Explorer is a recreation of the Explorer interface from Roblox Studio, designed to replicate how instances and hierarchies are displayed and interacted with.
The goal of this repository is to provide a simple, open source implementation that can be used for learning, experimentation, or integration into other Roblox projects.

## Features
* Instance hierarchy display
* Expand / collapse instance trees
* Explorer-style UI
* Designed to resemble the Roblox Studio Explorer

## Installation
1. Download or clone the repository
2. Insert **Explorer.luau** into your Roblox project
3. Run the script to initialize the Explorer UI

## Usage
Once installed, the Explorer will display the instance hierarchy similarly to the built-in Roblox Studio Explorer.
You can expand and collapse instances to navigate the structure.
